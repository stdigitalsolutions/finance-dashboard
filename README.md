📌 Overview
Professional single-file finance dashboard built with pure HTML/CSS/JS. Tracks expenses, converts currencies, and auto-saves data. South African Rand (ZAR) focused.

✨ Features
✅ Auto-Save - Data saved automatically to browser

✅ ZAR Currency - South African Rand as primary

✅ 150+ Currencies - Global converter with real rates

✅ Professional UI - Clean, responsive design

✅ Dark/Light Mode - Theme toggle

✅ Charts & Analytics - Visual financial data

✅ No Pre-filled Data - Start fresh every time

✅ Mobile Responsive - Works on all devices

🛠️ Tech Stack
HTML5 / CSS3 / Vanilla JavaScript

Chart.js for visualizations

Tailwind CSS via CDN

ExchangeRate-API for currency data

LocalStorage for data persistence
